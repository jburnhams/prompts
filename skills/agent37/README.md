A small community collection of Claude Code plugins. Only the code-review
relevant one is included here; the repo also has `generate-prompt-request`
and `yc-advisor` plugins, which are out of scope for this collection.

## Folders

- `local-review/` — reviews **uncommitted local changes** (staged +
  unstaged), explicitly modeled on Anthropic's official `code-review`
  plugin (see `../anthropic/code-review/`) but for pre-commit review
  instead of PR review.
