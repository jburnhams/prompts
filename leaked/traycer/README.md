# Traycer AI

- **Type**: Coding planning/execution agent (works alongside another
  coding agent) · **Vendor**: Traycer · **Status**: closed source (leaked)
- **Mirror source**: https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools/tree/main/Traycer%20AI (2026-07-10)

## Files
- `plan_mode_prompts.txt` / `plan_mode_tools.json` — "plan mode": breaks a
  task into phases before any execution.
- `phase_mode_prompts.txt` / `phase_mode_tools.json` — "phase mode":
  executes one phase of an existing plan.
