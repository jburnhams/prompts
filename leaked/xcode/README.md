# Xcode (Intelligence / Coding Assistant)

- **Type**: Coding agent (Xcode's built-in AI actions) · **Vendor**: Apple
- **Status**: closed source (leaked)
- **Mirror source**: https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools/tree/main/Xcode (2026-07-10)

Rather than one agent prompt, this is a set of prompts for individual
context-menu "actions" Xcode offers.

## Files
- `System.txt` — shared base system prompt.
- `DocumentAction.txt` — generate documentation comments.
- `ExplainAction.txt` — explain selected code.
- `MessageAction.txt` — write a commit message.
- `PlaygroundAction.txt` — generate a Swift Playground.
- `PreviewAction.txt` — generate a SwiftUI preview.
